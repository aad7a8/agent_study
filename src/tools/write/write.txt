Tool: write
Description: Write (overwrite) the full content of a file. Creates parent directories if needed.
Use this to create new files or completely rewrite an existing one.
For targeted changes to existing files, prefer the edit tool instead.

Args:
  path (str, required): Path to the file to write.
  content (str, required): Full content to write. Replaces any existing content.

Returns:
  str — confirmation message with line count.
